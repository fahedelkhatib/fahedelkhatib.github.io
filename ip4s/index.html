iPhone 4s-like Web Components

Last updated: 01/06/20

Summary

These are a set of webcomponents created using the webcomponent API based on the native widgets for early versions of iOS. A JavaScript file with all components can be found here, or can be downloaded seperately.

Buttons

There are 5 different colored basic buttons. All buttons are responsive, and you can add click events to them by using addEventListener.

HTML Markup Button
<ip-button-green>Green Button</ip-button-green> Green Button
<ip-button-red>Red Button</ip-button-red> Red Button
<ip-button-yellow>Yellow Button</ip-button-yellow> Yellow Button
<ip-button-grey>Grey Button</ip-button-grey> Grey Button
<ip-button-black>Black Button</ip-button-black> Black Button

Popup Alert Boxes

Popup alerts are created with <ip-popup-box>Popup Alert</ip-popup-box>

Popup Alert

By default, they create two buttons with default Confirm and Cancel text. However, this text can be set using the leftlabel and rightlabel properties of the tag. Additionally, a title can be set using the boxtitle property. For example, with the following markup: <ip-popup-box boxtitle="Alert Title" leftlabel="Hello" rightlabel="World">Popup Alert</ip-popup-box>

Popup Alert

To target the buttons, you can target the LeftButton and RightButton classes of the <ip-popup-box>Popup Alert</ip-popup-box> tag.

Toggle Button

A toggle button can be creating using <ip-toggle-button onvalue="ON" offvalue="OFF"></ip-toggle-button>

You can set the on and off values for the toggle with the onvalue and offvalue properties

To make the toggle button clickable, you can add an event listener on the toggle button tag itself.

Dropdown Buttons

Dropdown buttons consist of two tags, the <ip-button-dropdown-container></ip-button-dropdown-container> which will contain the dropdown buttons using the <ip-button-dropdown>Label</ip-button-dropdown> tags. For example, with the following markup: 

<ip-button-dropdown-container>
    <ip-button-dropdown first>Label</ip-button-dropdown>
    <ip-button-dropdown>Hello</ip-button-dropdown>
    <ip-button-dropdown>World</ip-button-dropdown>
</ip-button-dropdown-container>

Will produce the following dropdown buttons:

Label Hello World

Notice that the first button needs to include the blank property "first". To make the buttons clickable, you can add them on the dropdown buttons tags.

Pins

Currently there are 4 different pins you can create, with the following tags:

  • <ip-pin-blue-arrow></ip-pin-blue-arrow>
  • <ip-pin-blue-plus></ip-pin-blue-plus>
  • <ip-pin-green-plus></ip-pin-green-plus>
  • <ip-pin-red-minus></ip-pin-red-minus>

You can add click events on the tags themselves to make the pins clickable.

Notifications

Notifications are simply text boxes that you can use for notification text. They can be created with the <ip-notification>Hello World</ip-notification> tag:

Hello World

By default text is aligned to the left, but if you want to center it, you can simply add stylings to the tag. For example, here is markup to center the text: <ip-notification style="text-align: center;">Hello World</ip-notification>

Hello World

Scroll Buttons

Scroll buttons involve two tags, the <ip-scroll-button-container></ip-scroll-button-container> which will hold <ip-scroll-button>Label</ip-scroll-button>. For example, the following markup: 

<ip-scroll-button-container>
    <ip-scroll-button>Label 1</ip-scroll-button>
    <ip-scroll-button>Label 2</ip-scroll-button>
    <ip-scroll-button>Label 3</ip-scroll-button>
    <ip-scroll-button>Label 4</ip-scroll-button>
    <ip-scroll-button>Label 5</ip-scroll-button>
    <ip-scroll-button>Label 6</ip-scroll-button>
    <ip-scroll-button>Label 7</ip-scroll-button>
    <ip-scroll-button>Label 8</ip-scroll-button>
</ip-scroll-button-container>

Will produce:

Label 1 Label 2 Label 3 Label 4 Label 5 Label 6 Label 7 Label 8

You can add click events on the <ip-scroll-button>Label</ip-scroll-button> tags with addEventListener.

Phone Container

The last tag makes it easy to create a phone container that can be used to hold these other widgets. The <ip-phone-container pwidth="300px" pheight="400px"> tag creates the phone container, with the option to specify a width and a height for the container.

You can place other tags within this container to create your GUI reminiscient of the GUI on older versions of iOS

Summary

While the gradiant style of the widgets in the older iOS versions have become less popular these days, given the shift to the flat design style, the old iOS widgets had a charm to them, and still look good to this day.
