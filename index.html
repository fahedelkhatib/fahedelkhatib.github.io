My Projects

Below are a list of some of my personal projects. They are coded using a variety of languages and libraries, and I enjoy learning them more intimately along the way.


My Fleet of Discord Bots

I won't lie. This is definitely the most fun I have had with a project. I had to learn two separate api's and a new programming language. These bots do some fun things when they detect keywords in messages that are sent in a discord server. There are some useful commands too!

RSA Decoder/Encoder

This one is a work in progress. A beautiful mess! The end goal here is to make something lightweight, powerful, and user-friendly. Encryption and privacy should not be a luxury.

Memeparser

A deprecated python project of mine that systematically downloads memes in bulk using the reddit api. The original vision was to download memes automatically from any website, but I have relegated this project ot hackathons. My default hackathon headscratcher!
